# Serenity - Mindfulness & Wellness App

A beautiful, buildless web application for mental wellness featuring breathing exercises, a mood journal, and AI-powered guidance.

Built by [Teda.dev](https://teda.dev), the AI app builder for everyday problems.

## Features

*   **Breathing Exercises**: Visual guided animations (Box Breathing) to reduce stress.
*   **Mood Journal**: Local-first storage to track emotions securely.
*   **ZenGuide AI**: Client-side AI (WebLLM) for private, intelligent wellness advice.
*   **Responsive Design**: Fluid glassmorphism UI that works on desktop and mobile.

## Technology Stack

*   **HTML5/CSS3**: Modern layout with Tailwind CSS.
*   **jQuery**: DOM manipulation and state management.
*   **WebLLM**: In-browser Large Language Model inference (WebGPU required).
*   **LocalStorage**: Persistence for journal entries.

## Usage

1.  Open `index.html` to view the landing page.
2.  Click "Launch App" to enter `app.html`.
3.  Navigate through the sidebar to access tools.
4.  Note: The AI feature requires a WebGPU-compatible browser (Chrome 113+, Edge 113+). It will download the model (~1.6GB) upon first use of the AI tab.
